Latitude vs. X

The purpose of this assignment was to analyze the weather as you get closer to the equator. A OpenWeatherMap API was used to accumulate a dataset for observation. Several scatter plots were created to further asses the difference in temperature depending on the latitude.

Once the dataset was acquired, Python coding was used to generate several different charts to visualize the data. By using jupyter notebook and importing pandas and matplotlib, different graphs were created to visualize cloudiness, max temp, humidy and windspeed.
